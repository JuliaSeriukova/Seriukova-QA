DNS - Domain Name System - система для  получения информации о доменах, дает возможность находить сайт по его имени с помощью DNS-серверов. С помощью DNS мы можем перейти на интересущий нас сайт по имени домена.
Когда пользователь отправляет запрос на сервер, например, google.com, доменное имя преобразовывается в ip адрес (числовой идентификатор), понятные для машины.
С помощью IP адреса можно узнать откуда загружается страница сайта.
С помощью DNS сервера предоставляется доступ к системе доменных имен, какие ip адреса соответствуют домену. DNS сервер хранит записи доменов, а также кеширует записи других DNS серверов.

HTTP - это Hyper Text Transfer protocol, протокол передачи языка гипертекстовой разметки на основе ТСP/IP, набор стандартов для взаимодействия веб-браузеров и серверов, использует технологию клиент-сервер, в отличие от HTTPS нет механизма шифрования данных, информация передается в незащищенном виде, HTTP использует по умолчанию порт 80, передает данные открытым текстом, работает быстрее чем HTTPS. Существует несколько версий, сейчас в основном используется 1.1.

Версия 1.0. - использовал TCP соединение, требовал несколько обменов сообщениями, был медленным 
Версия 1.1. - конвейерная обработка, устойчивое соединение для передачи данных HTML и произвольных данных, существовала проблема блокировки начала строки (устранена в версии 2)
Версия 2 - вторая версия на базе протокола SPDY прикладного уровня, совместим с функциями 1.1, более быстрая загрузка, несколько запросов в одном соединении, первый бинарный протокол

HTTPS отличает secure, т е это защищенный протокол передачи, это расширение протокола HTTP, которое использует шифрование путем SSL сертификата и TLS, более безопасный т к позволяет установить защищенное соединение, использует по умолчанию порт 443, передает зашифрованные защищенные данные, работает медленнее чем HTTP. HTTPS нужен для сайтов, работающих с конфиденциальной информацией (например, личные данные пользователя, платежная информация). 
Поисковые выдачи приоритизируют сайты с https.
